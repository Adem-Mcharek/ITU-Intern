About WebTV Processor

An AI-powered platform for processing UN WebTV content into searchable transcripts

Open Source

Built with Flask, Bootstrap, and modern web technologies

AI-Powered

Uses OpenAI Whisper and Google Gemini for accurate results

Scalable

Background processing with Celery for high-volume usage

Features
  • Audio extraction from WebTV streams
  • High-accuracy AI transcription
  • Speaker identification and separation
  • Multiple output formats (TXT, SRT)
  • Searchable meeting database
  • Real-time processing status
Technology Stack
Backend
Flask, SQLAlchemy, Celery
Database
SQLite (dev), PostgreSQL (prod)
Frontend
Bootstrap 5, HTMX, Vanilla JS
AI/ML
OpenAI Whisper, Google Gemini
Media
yt-dlp for video processing

Redis + Celery workers
